# 财务管家 (SalaryManager)

> 3 秒记一笔，一张图看懂你的钱去哪了。

## 产品简介

**财务管家**是一款面向 18-35 岁年轻用户的个人记账工具，致力于解决传统记账 App"记着记着就放弃"的痛点。

### 核心价值

| 问题 | 财务管家的解法 |
|------|--------------|
| 操作繁琐，坚持不下去 | 3 步（金额 → 分类 → 保存）、5 秒内完成记账 |
| 统计图表看不懂 | 首页一张图展示本月支出总额、分类占比、预算进度 |
| 分类太多选择困难 | 仅 8 个默认分类，智能记忆上次选择 |
| 数据分散在各个支付渠道 | 手动记录（V1.0）→ 自动同步（V2.0） |

### 产品特色

- 🌱 **春日绿色主题** — 柔和护眼，消除记账焦虑感
- ⚡ **极简交互** — 单笔记账 < 5 秒
- 📊 **一眼就懂** — 首页零操作获取核心财务信息
- 🔒 **离线可用** — 本地数据存储，无需网络
- 👤 **多用户支持** — 独立账号，数据完全隔离

## 快速启动

### 前端

```bash
cd 004.前端代码/salary-vue

# 安装依赖
npm install

# 开发模式（热重载）
npm run dev

# 构建生产版本
npm run build

# 预览生产构建
npm run preview
```

前端访问地址：`http://localhost:5173`

### 后端

**环境要求：** JDK 17+

```bash
cd 006.后端代码/backend

# 开发模式运行
./mvnw spring-boot:run

# 或者打包后运行
./mvnw clean package
java -jar target/salary-manager-1.0.0.jar
```

后端端口：`http://localhost:8080`

### 数据库

需要本地运行 MySQL 8.0+，并创建数据库：

```sql
CREATE DATABASE salary_manager DEFAULT CHARACTER SET utf8mb4;
```

数据库配置在 `backend/src/main/resources/application.yml` 中。

## 功能模块

| 模块 | 功能 |
|------|------|
| 📝 记账 | 快速记账、编辑/删除、备注、日期选择 |
| 📊 首页看板 | 月度支出总额、分类占比环形图、7天趋势、预算进度 |
| 🏷️ 分类管理 | 8 个预设分类 + 自定义分类（上限 10 个） |
| 💰 预算管理 | 月度总预算、分类预算、80%/100% 预警 |
| 📋 记录列表 | 按月查看、分类筛选、左滑删除、右滑编辑 |
| 📤 数据导出 | CSV 格式导出（UTF-8 with BOM，兼容 Excel） |
| ⚙️ 设置 | 预算设置、分类管理、深色模式、数据备份 |
| 🔐 用户系统 | 注册/登录、JWT 会话、密码 SHA-256 哈希 |

## 设计规范

- **设计原则**：速度优先（< 5 秒记账）、一眼就懂、离线可用、安全感、美观舒适
- **性能目标**：冷启动 ≤ 1.5s、记账保存响应 ≤ 200ms、10000 条记录首页加载 ≤ 1s
- **无障碍**：WCAG AA 级色彩对比度、触控区域 ≥ 44×44px
