## . 换个角色思考（技术同事视角）

skill的网站：https://www.skills.sh/

#### 产品岗专用指令式 Skill

1. ```
   /prd-generate
   ```

   介绍：输入业务需求，自动生成完整标准化 PRD 文档，包含功能逻辑、异常分支、字段定义、验收标准

2. ```
   /prd-check
   ```

   介绍：上传已有 PRD，自动巡检缺失边界、权限漏洞、遗漏埋点、逻辑矛盾等需求缺陷

3. ```
   /mermaid-flow
   ```

   介绍：口述业务流程，一键生成可直接复制的泳道图、流转流程图 Mermaid 代码

4. ```
   /user-story-case
   ```

   介绍：根据页面功能批量产出用户故事、正向 / 反向验收测试用例

5. ```
   /track-event
   ```

   介绍：录入页面交互点位，生成完整埋点清单、事件名称、上报属性、统计口径

6. ```
   /metric-def
   ```

   介绍：输入业务目标，输出 KPI 指标、统计维度、计算公式、数据筛选规则

7. ```
   /proto-doc
   ```

   介绍：描述页面布局、弹窗跳转逻辑，生成可交付 UI / 前端的原型交互说明文档

8. ```
   /task-split
   ```

   介绍：拆分大型需求为前后端、测试细分任务，标注依赖关系与工作量说明

9. ```
   /meeting-minutes
   ```

   介绍：粘贴评审聊天文本，自动提取问题、待办、责任人、排期，结构化输出纪要

10. ```
    /req-tech-convert
    ```

    介绍：将口语化业务需求转换为开发可读的标准化技术需求描述

#### 开发岗专用指令式 Skill

1. ```
   /backend-crud
   ```

   介绍：提供 MySQL 建表语句，一次性生成 Entity、Mapper、Service、Controller、接口文档全套代码

2. ```
   /redis-lock-cache
   ```

   介绍：生成生产可用 Redis 缓存封装、分布式锁代码，自带防穿透 / 击穿 / 雪崩方案

3. ```
   /rocketmq-code
   ```

   介绍：输出 RocketMQ 生产者、消费者、事务消息、死信队列完整业务模板

4. ```
   /mongo-replica
   ```

   介绍：生成 Mongo 单 / 多节点副本集初始化、权限配置、Docker 启动 Shell 脚本

5. ```
   /global-exception-valid
   ```

   介绍：生成项目统一返回体、JSR303 参数校验、全局异常捕获处理代码

6. ```
   /systemd-service
   ```

   介绍：输入 Java 服务启动参数，自动生成带独立 JDK、自动重启的 systemd 配置文件

7. ```
   /gradle-build
   ```

   介绍：优化 git 拉取、打包、后台启停完整命令，增加构建失败校验逻辑

8. ```
   /sql-optimize
   ```

   介绍：解析慢 SQL 语句，给出索引优化、语句重构方案并输出改造代码

9. ```
   /junit-test
   ```

   介绍：基于 Service/Controller 业务代码，自动生成覆盖正常、异常场景单元测试
